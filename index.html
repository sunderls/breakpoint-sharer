breakpoint sharer

Breakpoint Sharer is just a simple Chrome Extension that enables you to import/export debugging breakpoints, to explain/learn code logic better

Chrome Extension

install here : https://chrome.google.com/webstore/detail/hkbgnljbklddbfmkmeaoolmdflbbofmi

usage

Basically it is similar to original Chrome DevTool, with breakpoint import/export function, but lacking many many many other features.
